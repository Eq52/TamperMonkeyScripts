# 关于XJ解析

> 此脚本为人工提供具体思路并由AI们共同创作...



**使用方法**

使用脚本扩展安装此脚本后即可在刷新后的B站具体视频/番剧页面中看到"📺"按钮，点击📺按钮即可使用...



**功能大致预览**

1. 设置面板 ✅

2. 自动解析设置 ✅

3. 广告清除(可能无效)  ❌

4. 番剧接口自定义 ✅

5. 视频下载 ✅

   > 目前仅支持BV号

6. 清晰度提升方案 ✅

   > 点击NX 极速解析按钮即可 ; 目前仅支持BV号
